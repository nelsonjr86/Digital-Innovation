## :information_source: Sobre

Recriando a página inicial do Netflix.

Nesse projeto, era recriar a interface do Netflix, foi abordado o carousel para recriar o efeito para passar as imagens.
Utilizando tecnologias simples como HTML5, CSS3 e JavaScript.


## :interrobang: Motivo

O projeto faz parte do Bootcamp HTML Web Developer da Digital Innovation One.

Aprenda a programar páginas de internet e desenvolver websites utilizando HTML, CSS e JavaScript para iniciar sua trajetória profissional em desenvolvimento web front-end.
